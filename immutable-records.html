Tamper-Proof Records on the BSV Blockchain.

Secure your business data with immutable anchoring for compliance, audits, and trust.

What It Is

Immutable Records is a service that lets you hash your business records—like invoices, contracts, and purchase orders—using SHA-256, anchor them to the BSV blockchain, and store the transaction ID in Supabase for easy retrieval and verification.

How It Works

🔒

Hash

Generate a secure SHA-256 hash of your document or data.

⚓

Anchor

Embed the hash into a BSV transaction for permanent, timestamped proof.

✅

Verify

Confirm the record's integrity anytime using the stored txid.

API Endpoints

/anchor

POST /anchor { "hash": "your-sha256-hash" } → { "txid": "bsv-txid" }

/verify

GET /verify?txid=bsv-txid → { "verified": true, "timestamp": "..." }

/hash

POST /hash { "data": "your-data" } → { "hash": "sha256-hash" }
